# Spaceflight Simulator
## What is it?
Spaceflight simulator is a fun website where people can simulate their own rocket launch to the ISS!

## Inspiration
I have a lot of interest in space exploration. In simulations, the key thing is that even one factor can mess up the success chances. 
In this simulation, there is randomized weather, risks with fuel, and more.

## Built with:
* HTML
* CSS
* Javascript
